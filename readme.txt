## Project Overview
Modern cloud systems produce millions of logs daily across microservices.  
Manually analyzing these logs is inefficient and error-prone.  

**Intelligent Log Insights** is a fully automated, AI-powered log analysis system that:
- Ingests logs in real time.
- Processes and stores them in Azure Cosmos DB.
- Summarizes insights automatically.
- Sends alerts through Azure Logic Apps.
- Exposes analytical APIs via FastAPI.
- Enables **semantic log search** using vector embeddings & Gemini AI.
